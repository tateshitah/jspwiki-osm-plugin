Abstract#

以前は住んでいたということで旅行の観点からはまとめていませんでしたが、2017年9月に行ってきましたので、その観点からもこれまでの情報をまとめたいと思います。

Foods#

  • Restaurant in DC
DCエリアで食べておいしかったレストランなどを記録しておきたいと思います。

Hotel#

  • Virginian Suites
I often use this hotel.(2008年5月、2009年1月、2017年9月） It locates near Rosslyn station, Virginia, so little bit more reasonable than hotels in DC. There is no restaurant in this hotel, but we can find some restaurants near Rosslyn station and if you use metro, you can go to centre of DC (North West area) easily.
  • Tabard Inn
2009年8月に妻と泊まりました。とても古めかしく、洗練されていて、スタイリッシュで、この国には珍しくサービスも行き届いていて、素晴らしいホテルでした。ブランチでも有名らしいです。
